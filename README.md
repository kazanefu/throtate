# Throtate

**回って飛んでゴールを目指す2Dアクションゲームです。**

## 操作方法

- `Space`: **回る** と **飛ぶ**　を切り替える
- `R`: リスポーン (チェックポイントから再スタート)
- `Escape`: ホーム画面に戻る
- `Arrow Keys`: 回転方向と回転軸の切り替え
  - `Left`: 水色の回転軸で反時計回りに回る
  - `Right`: ピンクの回転軸で時計回りに回る
  - `Up`: ピンクの回転軸で反時計回りに回る
  - `Down`: 水色の回転軸で時計回りに回る

## ルール

- プレイヤーは回転と飛行を切り替えながら、ゴールを目指します。

## オブジェクト

- **プレイヤー**: 回転と飛行を切り替えることができるキャラクター。 見た目: **隕石**
- **チェックポイント**:　触れたときにリスポーンポイントを更新するオブジェクト。 見た目: **水色**
- **ゴール**: 触れたときにゲームクリアとなるオブジェクト。 見た目: **ピンク**
- **デスプロック**: 触れたときに **死亡数** を+1してリスポーンポイントに戻されるオブジェクト。 見た目: **赤色**
- **タレット**: 小さな **デスプロック** を定期的に発射するオブジェクト。 見た目: **オレンジ**
- **スマッシュブロック**: 一定速度以上で衝突することで破壊できるオブジェクト。 見た目: **ヒビの入った灰色**
